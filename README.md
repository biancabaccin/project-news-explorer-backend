# Project News Explorer Backend

Este projeto faz parte da etapa de Back-end do programa de Web Development da TripleTen, com foco na construção de uma API segura e estruturada utilizando Node.js, Express e MongoDB.

Nesta fase, o objetivo é desenvolver a lógica de autenticação, criação de rotas protegidas, integração com banco de dados e implantação da API em produção.

## Principais Recursos Utilizados:

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- Estrutura REST API
- Middleware customizado
- Variáveis de ambiente com dotenv
- ESLint

### Autenticação e Segurança

- JWT (JSON Web Token)
- bcryptjs para criptografia de senhas
- Rotas protegidas por autenticação
- Validação de dados com Celebrate/Joi
- Controle de acesso por usuário
- Middleware centralizado de erros

### Logs

- request.log → registra todas as requisições da API
- error.log → registra erros da aplicação
